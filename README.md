# Netflix Content Analysis Dashboard (Tableau)

A Tableau dashboard project that explores Netflixâ€™s catalog to understand content distribution, country and genre trends, ratings breakdown, and release patterns over time.

## What this project shows
- Total titles, movies vs TV shows split, and rating distribution
- Content by country and genre (top categories and regions)
- Release year trends and additions over time
- Filters to drill down by type, rating, country, and release year

## Files in this repo
- `Netflix Dashboard.twb` - Tableau workbook
- `netflix_titles.csv` - dataset used for the analysis

## How to view
1. Open `Netflix Dashboard.twb` in Tableau Desktop.
2. If Tableau prompts for a data source, connect it to `netflix_titles.csv`.

## Tools used
Tableau, CSV dataset
